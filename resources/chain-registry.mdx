Neutron is registered in the [Cosmos Chain Registry](https://github.com/cosmos/chain-registry), providing standardized information about the chain for wallets, explorers, and tools in the Cosmos ecosystem.

## Registry Information

| Resource | URL |
|----------|-----|
| Chain Registry | [chains.cosmos.directory/neutron](https://chains.cosmos.directory/neutron) |
| Chain.json | [chains.cosmos.directory/neutron/chain](https://chains.cosmos.directory/neutron/chain) |
| Assetlist | [chains.cosmos.directory/neutron/assetlist](https://chains.cosmos.directory/neutron/assetlist) |
| Validator Registry | [validators.cosmos.directory/chains/neutron](https://validators.cosmos.directory/chains/neutron) |

## Using Chain Registry Information

Chain Registry information can be used to:

1. Configure wallets to support Neutron
2. Set up nodes with correct parameters
3. Integrate Neutron into ecosystem tools and explorers
4. Access standardized information about IBC connections

For developers building on Neutron, this information is essential for ensuring compatibility with ecosystem tools.
