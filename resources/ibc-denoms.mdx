## Using IBC Denoms

To specify an IBC token in transactions or queries, use the full IBC denom format. For example:

```bash
# Query ATOM balance
neutrond query bank balances neutron123... --denom=ibc/C4CFF46FD6DE35CA4CF4CE031E643C8FDC9BA4B99AE598E9B0ED98FE3A2319F9

# Send TIA tokens
neutrond tx bank send \
  my-wallet recipient-address \
  1000000ibc/773B4D0A3CD667B2275D5A4A7A2F0909C0BA0F4059C0B9181E680DDF4965DCC7 \
  --chain-id neutron-1 \
  --gas auto \
  --gas-adjustment 1.3
```

## IBC Relayers

IBC tokens are transferred and maintained by IBC relayers that process packet data between Neutron and other chains. The primary relayers for Neutron include:

1. Neutron Foundation relayers
2. Community relayers (Lavenderfive, Notional, Polkachu, etc.)
3. Third-party commercial relayers
