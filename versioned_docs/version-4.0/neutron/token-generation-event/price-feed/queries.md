# Queries

## `get_config`

```json
{
  "get_config": {}
}
```

Returns the contract's configuration.


## `get_error`

```json
{
  "get_error": {}
}
```

Returns an error that happened during request of a new prices.

## `get_rate`

```json
{
  "get_rate": {}
}
```

Returns price rate of the given symbols.
