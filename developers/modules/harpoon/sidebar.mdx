## Introduction

The Harpoon module allows CosmWasm smart contracts to subscribe to staking module hooks. This functionality is particularly important for tracking historical voting power and implementing sophisticated governance systems.

## Documentation Structure

The Harpoon module documentation follows the [Divio documentation system](https://documentation.divio.com/), which organizes content into four distinct types:

1. **Overview** — High-level description of the module (tutorials)
2. **Explanation** — Conceptual explanations of the module's architecture and components
3. **How-To** — Practical step-by-step guides for completing specific tasks
4. **Reference** — Technical information, APIs, and message specifications
