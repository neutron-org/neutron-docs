# Harpoon Module

The Harpoon module allows CosmWasm smart contracts to subscribe to staking module hooks. This functionality is particularly important for contracts that need to track historical voting power from the staking module, enabling sophisticated governance systems and token distribution mechanisms.

For detailed documentation, please visit the [Harpoon Overview](/developers/modules/harpoon/overview).
