## Technical Questions

### I am a developer who builds on Neutron. How can I get in touch with the dev team?

There are three official channels for technical communication:

- <a href="https://t.me/+Q6C_kb0xLaoxYzE1" target="_blank" className="text-primary hover:underline">Neutron Builders Announcements</a>: Official announcements channel
- <a href="https://discord.com/invite/bzPBzbDvWC" target="_blank" className="text-primary hover:underline">Neutron Builders Chat</a>: Join our Discord server for real-time technical discussion
- <a href="https://t.me/+GK5KVutIvQhhNmNi" target="_blank" className="text-primary hover:underline">Neutron Dev Blog</a>: Technical notes from the development team

### I think I've found a vulnerability, what should I do?

Check out our bug bounty program on <a href="https://immunefi.com/bug-bounty/neutron" target="_blank" className="text-primary hover:underline">Immunefi</a>.

### Where is the block explorer?

We recommend using the following block explorers:

- **Mintscan Explorer**:
  - <a href="https://mintscan.io/neutron" target="_blank" className="text-primary hover:underline">Mainnet (neutron-1)</a>
  - <a href="https://mintscan.io/neutron-testnet" target="_blank" className="text-primary hover:underline">Testnet (pion-1)</a>
- **Celatone Smart Contract Explorer**: Provides a great UI to upload, query and execute contracts.

### Where can I download the latest snapshots?

If you're looking to spin up your own archive node, our <a href="https://cosmos-snapshots.neutron.org/" target="_blank" className="text-primary hover:underline">snapshot service</a> provides a convenient solution, enabling you to skip syncing from scratch. We also offer small pruned snapshots for quick server bootstrapping.

### Where can I get the public REST and RPC nodes?

You can check the Cosmos Chain Registry for public endpoints:

- <a href="https://github.com/cosmos/chain-registry/tree/master/neutron" target="_blank" className="text-primary hover:underline">Mainnet (neutron-1)</a>
- <a href="https://github.com/cosmos/chain-registry/tree/master/testnets/neutrontestnet" target="_blank" className="text-primary hover:underline">Testnet (pion-1)</a>

For a list of available endpoints, see our <a href="/resources/endpoints" className="text-primary hover:underline">Endpoints Reference</a>.

### How can I check the current status of Neutron's RPCs?

For the most current information on the status of our RPCs and any scheduled maintenance, please visit our <a href="https://status.neutron.org/" target="_blank" className="text-primary hover:underline">Neutron Status</a> page. This page provides real-time updates and alerts regarding the operational status of all Neutron services.

### What CosmWasm version is used?

We are using a patched 0.51.0 version of wasmd.

### Where is the testnet faucet?

You can request tokens from the Testnet (pion-1) Faucet through two channels:

1. **Discord**: Post in the #testnet-faucet channel on <a href="https://discord.com/invite/bzPBzbDvWC" target="_blank" className="text-primary hover:underline">Neutron's Discord server</a> with the command: `$request <NEUTRON-ADDRESS>`
2. **Telegram**: Use the <a href="https://t.me/NeutronTestnetFaucet_bot" target="_blank" className="text-primary hover:underline">Telegram Faucet Bot</a> with the command: `/request <NEUTRON-ADDRESS>`

Where `<NEUTRON-ADDRESS>` is a `neutron1******` generated address.
