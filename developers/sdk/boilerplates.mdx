## Template Categories

### Full-Stack dApp Templates
Complete application templates that include both frontend and smart contract components, ready for customization and deployment.

### Integration Libraries
Specialized tools and documentation for integrating with specific Neutron ecosystem protocols and platforms.

### Testing & Development Tools
Local testing environments and development utilities to streamline your build process.

## Quick Start

Choose a template based on your project needs:

1. **Building a general dApp?** → Start with [NeutronTemplate](/developers/sdk/neutron-template)
2. **Integrating with trading platforms?** → Use [Bull vs. Bear tools](/developers/sdk/bullbear-integration)
3. **Need local testing?** → Check out the testing utilities included in each template

## Community Contributions
