Neutron provides developers with a powerful set of native modules, features, and tools to build sophisticated DeFi applications with unparalleled capital efficiency and atomic composability.

## Documentation Structure

Our developer documentation follows the [Divio documentation system](https://documentation.divio.com/), organizing information into four types:
- **Tutorials**: Learning-oriented guides for beginners
- **Explanations**: Understanding-oriented conceptual information
- **How-to Guides**: Problem-oriented practical steps
- **Reference**: Information-oriented technical descriptions
