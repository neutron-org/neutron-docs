# Queries

## `config`

```json
{
  "config": {}
}
```

Returns information about contract's configuration

## `state`

```json
{
  "state": {}
}
```

Returns information about contract's state.

## `user_info`

```json
{
  "user_info": {
    "address": "neutron..."
  }
}
```

Returns info about user position.
